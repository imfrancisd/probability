# Engineering

This contains all the build scripts and tools necessary to build the project.

## Episode01

Build Episode01 with the following command
```
REM From cmd.exe
powershell.exe -noninteractive -noprofile -executionpolicy remotesigned -file buildEpisode01.ps1
```
or just run the script from PowerShell
```
### From powershell.exe
.\buildEpisode01.ps1
```
